Installation de ROS --------------------------------------------

Drivers Nvidia:
Avant tout, tester si le pc s'éteint correctement. Si non, installer les drivers nvidia via:
sudo apt-get install nvidia-384
ou
sudo apt-get install nvidia-430

https://ubuntu-mate.community/t/is-there-a-ubuntu-mate-for-raspberry-pi-4/19943/4

Puis changer cmdline.txt (dans /boot) en

https://thepihut.com/blogs/raspberry-pi-tutorials/19668676-renaming-your-raspberry-pi-the-hostname
etape 1: modification réseau -> nouvelle connection -> vers ordinateur (IPv4) et ignorer (IPv6)
etape 2: hostname -I, puis avec l'ip trouvée faire sudo nmap -sP -PI -PT *IP*/24
etape 3: faire sudo nano /media/magnus/PI_ROOT/etc/hostname et noter le nom
etape 4: sudo ssh prmraspi@*IP trouvée avec nmap* et entrer le MdP (ici sudo ssh -X prmraspi@10.42.0.201)
etape 5: gnome-terminal & pour avoir plusieurs terminaux
